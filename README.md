# Sere Innovations – Smart Incubator Landing Page

A responsive, farmer‑friendly landing page for Sere Innovations’ smart egg incubator.  
Built as part of the Frontend Developer Intern assessment for NayaGrowth.



---

## 📌 Project Overview

Sere Innovations helps small poultry farmers hatch their own chicks using an affordable, easy‑to‑use smart incubator. This landing page clearly communicates the product, builds trust, and encourages farmers to enquire or book a demo.

The design follows the brand’s **simple, trustworthy, human, grounded** personality – no hype, no fake testimonials, just honest communication.

---

🛠️ Tech Stack
⚛️ React – Component-based UI
🎨 Tailwind CSS – Utility-first responsive styling
🔀 React Router DOM – Client-side routing
🚀 Vercel – Deployment

## 🚀 Features

- ✅ Fully responsive (mobile‑first, works on all devices)
- ✅ Semantic HTML + accessible components
- ✅ Smooth scroll to sections
- ✅ Auto‑rotating reviews carousel (6 farmer reviews, realistic names)
- ✅ Lead capture form + separate booking form for product intro
- ✅ Multi‑page structure:
  - Home (landing page)
  - About Us
  - Contact Us
  - Privacy Policy
  - Help Center
  - Farmers’ Guide
  - Warranty Info
- ✅ Animated SVG illustrations (floating incubator, egg bounce, hatching chick, pulsing temp)
- ✅ Scroll restoration on route change (`ScrollToTop` component)
- ✅ No fake testimonials, no invented awards – honest trust signals

---

## 📄 Sections Included (Landing Page)

The homepage contains all required sections from the brief:

1. **Navbar** – Logo, navigation links, mobile menu, CTA button
2. **Hero** – Headline, description, CTA, animated SVG incubator
3. **Problem** – Four key problems: dependency, high cost, loss of control, farm difficulties
4. **Solution** – How Sere Innovations solves each problem (two‑column layout)
5. **Features** – Six features (temp, humidity, auto turning, easy use, affordable, farmer‑friendly)
6. **Benefits** – Five benefits (control, independence, cost efficiency, simpler ops, income)
7. **Trust** – Built from real farmer needs, field‑grounded, no fake claims
8. **Audience** – Primary + secondary audiences with characteristics
9. **Reviews** – Auto‑rotating carousel of six farmer testimonials (realistic names & villages)
10. **CTA** – Four action buttons + enquiry form + booking form (toggles)
11. **Footer** – Brand, quick links, social icons, contact placeholders

## 🧪 Setup & Installation

2. **Install dependencies**
   npm install
3. **Start development server**
   npm start
4.**Build for production**
   npm run build
